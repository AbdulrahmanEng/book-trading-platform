# Book Trading Platform

A website for trading books.

![Preview](./preview.png "Preview")

## Tools

* Node.js
* Express.js
* Passport
* Auth0
* Pug
* Axios
* MongoLab
* Mongoose

## Instructions

1. Configure .env file with required variables.
2. Install packages.
3. Run `npm start`.
4. Register and login with email only.
